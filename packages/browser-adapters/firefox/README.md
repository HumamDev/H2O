# packages/browser-adapters/firefox — FUTURE PLACEHOLDER

Future Firefox-specific browser-API adapter — wraps `browser.*` (and
where needed the `chrome.*` polyfill) WebExtension APIs into a
versionable package consumed by per-host Firefox extensions under
`apps/extensions/<host>/firefox/`.

Not yet implemented. Companion to `packages/browser-adapters/chrome/`
(also a future placeholder). Firefox support is deferred until the
ChatGPT/Claude/Gemini host-adapter surface is stable enough to share.

Until then this README is the only tracked content; no package.json,
no source, no workspace entry.
