# R4.7 Investigation — Frozen Plan

This file freezes the R4.7 plan as approved at R4.7.1. Any later
deviations should land as separate amendments to this file, not as
silent inline edits.

## Premise

R4.6.4 committed `ee144af` shipped the default flag flip:

```
library.nativeWorkspaceUi:     false  (Native Library UI hidden by default)
library.nativeOrganizationUi:  false  (Native sidebar org sections hidden by default)
library.nativeCaptureOnlyMode: true   (Native is now capture-only)
```

The deprecation flags now resolve to hidden by default via
`NATIVE_FLAG_DEFAULTS` in 0F1k's `ensureFlags()`. Capture / save /
link / tag extraction continue to work unchanged.

After R4.6.4 soaked in production (zero capture-path errors, banner-
restore click rate at noise level, no support escalations), R4.7
physically removes the now-dormant UI code.
