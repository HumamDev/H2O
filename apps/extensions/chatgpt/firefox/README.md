# apps/extensions/chatgpt/firefox — FUTURE PLACEHOLDER

Future Firefox WebExtension build target for `chatgpt.com`.

Not yet implemented. The active ChatGPT extension is Chrome MV3 only,
built into `apps/extensions/chatgpt/chrome/<variant>/` by
`tools/product/extensions/chatgpt/chrome/build-chrome-live-extension.mjs`.

This folder reserves the spot for a future Firefox MV3 / WebExtension
sibling build when Firefox support is added. Until then this README is
the only tracked content here; the folder contains no build pipeline,
no source, and no manifest.
