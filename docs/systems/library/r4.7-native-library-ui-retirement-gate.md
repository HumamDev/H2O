## Rollback Procedure

Per-file rollback:

1. Open the relevant `retired-features/native-library-ui/<module>/`
   `extracted-from-*.md` file.
2. Restore the documented block from the matching archived `.js` file
   into the original Native source location.
3. Rerun the validator matrix below.

Per-slice rollback:

```bash
git revert <R4.7.N commit hash>
```

Full R4.7 rollback:

1. Revert R4.7.6 through R4.7.2 in reverse order.
2. Revert R4.7.1 only if the archive scaffolding itself must be
   removed.
3. Rerun all validators.

Restore flags warning:

`library.nativeWorkspaceUi` and `library.nativeOrganizationUi` still
exist for diagnostics, but setting them to `true` no longer restores
retired UI. The runtime code they once gated has been physically
removed. Rollback is the supported restore path.
